# 🧠 Title

**FreePay — Crypto Payment Links**

## 🚨 Problem

Receiving global payments is complex, slow, and requires intermediaries.

## 💡 Solution

FreePay enables anyone to create a payment link and receive crypto instantly.

## ✨ Features
- Create payment links
- Share via link / WhatsApp
- Pay using wallet
- Transaction confirmation

## ⚙️ Tech Stack
- Next.js
- TypeScript
- Tailwind + shadcn
- Starkzap SDK

## 🔌 Starkzap Usage
- Wallet connection
- Transaction execution
- Gasless payments (if used)

## 🚀 How it works
1. **Generate link**: Enter your target address and the execution amount to generate an instant payment vector. 
2. **Share**: Distribute the URL or scan the dynamic QR code directly from the interface.
3. **Pay**: Ensure seamless execution via Starknet.
